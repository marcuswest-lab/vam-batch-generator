## Generation Rules

1. **VARIABLE ISOLATION (critical for proper A/B testing):**
   - **Each batch tests ONE variable. All other variables are held constant.**
   - If **Variation Type = Lead** → ALL 5 videos use the SAME body script, CTA, talent direction, and visual treatment. Only the hook/opening changes.
   - If **Variation Type = Pattern Interrupt** → ALL 5 videos use the SAME lead, body, and CTA. Only the visual/audio pattern interrupt changes.
   - If **Variation Type = Graphic Overlay** → ALL 5 videos use the SAME lead, body, and CTA. Only the graphic/text overlay treatment changes.
   - If **Variation Type = Body** → ALL 5 videos use the SAME hook, CTA, and talent direction. Only the body talking points change.
   - If **Variation Type = CTA** → ALL 5 videos use the SAME hook, body script, and talent direction. Only the CTA changes.
   - The constant elements should be explicitly written out in the first creative, then referenced as "Same as Video 1" for subsequent creatives. Only the variable being tested gets new content.

2. **Apply 70-20-10 ONLY to the variable being tested (the held-constant variables do NOT change):**
   - Creatives 1-3 (70%): Variations of proven winners (Ialmostdidn't, IThoughtCostMore, IBudgetedMore, TruckInsured, TruckonTime)
   - Creative 4 (20%): Adapted from another niche's testimonial format
   - Creative 5 (10%): Experimental new angle
   - **Important:** 70-20-10 applies to the TESTED variable only. All non-tested components (body, CTA, talent direction, visual treatment) are literally identical across all 5 videos.

3. **UGC testimonials must dominate** — at least 4 of 5 should be UGC direct-to-camera. One slot can test Problem-Solution Lead format.

4. **Each hook script must feel natural and unscripted** — conversational tone, not polished marketing. Write talking points, not teleprompter scripts.

5. **DO NOT use:** Animated truck formats, holiday themes, educational/explainer formats, polished/branded production, ChatGPT conversation format

6. **Provide 2-3 hook variations per creative** (when testing Leads) — talent picks the most natural delivery

7. **Assign proper naming using Creative Tracker formula:** `[Angle Name] - [Lead Type] [Variation Type]` — e.g., Lead: `Done For You - Problem-Solution Lead`, Body: `Real Trucks Real Prices - Problem-Solution Body`. **Keep Angle Names SHORT** (2-4 words, natural casing). The number (#) and VID are auto-generated by the tracker.

8. **Dropdown constraints — use ONLY these values:**
   - Video Type: **Ad Video** (default for Meta ads)
   - Variation Type: **Lead** / **Pattern Interrupt** / **Graphic Overlay** / **Body** / **CTA** — ALL creatives in a batch should have the SAME variation type
   - Lead Type: **Offer** / **Promise** / **Problem-Solution** / **Secret** / **Proclamation** / **Story**
   - Awareness Level: **Most Aware** / **Product Aware** / **Solution Aware** / **Problem Aware** / **Unaware**
   - Status: **Ready For Internal**

## Post-Generation Checklist

After generating the batch, verify:
- [ ] **VARIABLE ISOLATION:** Only ONE variable changes across all 5 videos. All other components (hook, body, CTA, talent direction) are identical for the non-tested variables.
- [ ] All 5 creatives have the SAME Variation Type (all "Lead", all "Body", etc.)
- [ ] At least 4 of 5 are UGC format
- [ ] No losing patterns used (no animated trucks, holiday themes, educational format)
- [ ] Hook scripts sound natural/conversational, not polished
- [ ] Each creative has 2-3 hook variations for talent to choose (Hook batches only)
- [ ] Awareness levels are distributed across the batch
- [ ] Each has a clear data-driven rationale tied to winning ads
- [ ] Talent direction specifies authentic/iPhone-style (not produced)
- [ ] Naming follows Creative Tracker formula: `[Angle Name] - [Lead Type] [Variation Type]` (tracker adds # and VID)
- [ ] Variation Type uses ONLY: Lead / Pattern Interrupt / Graphic Overlay / Body / CTA
- [ ] Lead Type uses ONLY: Offer / Promise / Problem-Solution / Secret / Proclamation / Story
