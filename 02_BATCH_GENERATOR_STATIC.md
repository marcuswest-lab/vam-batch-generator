## Generation Rules

1. **VARIABLE ISOLATION (critical for proper A/B testing):**
   - **Each batch tests ONE variable. All other variables are held constant.**
   - Choose the best visual style based on performance data and hold it constant across ALL 5 creatives. Only the tested element (Hook, Body, CTA, etc.) changes between creatives. Design notes should be identical or near-identical across all 5.
   - This is non-negotiable. If you change two variables at once, you cannot attribute which one drove performance.

2. **Apply 70-20-10 ONLY to the variable being tested (the held-constant variable does NOT change):**
   - Creatives 1-3 (70%): Variations of proven winners from the Winning Hooks Library
   - Creative 4 (20%): Adapted from another service industry (home services, insurance, car dealers, travel)
   - Creative 5 (10%): Experimental — hasn't been tested
   - **Important:** 70-20-10 applies to the TESTED variable only. The visual style is identical across all 5 — you do NOT vary the visual to reach 70-20-10 on visuals.

3. **Each creative must use a DIFFERENT verbal hook type** (when testing Hook variation type): Labels, Questions, Conditionals, Commands, Statements, Lists/Steps, Narratives, Exclamations

4. **DO NOT use any element from the Confirmed Losing Patterns** (no StickyNote, EducationalLayout, HoldingSign, holiday themes, geographic targeting, carousel, etc.)

5. **Assign proper naming using Creative Tracker formula (depends on Variation Type):**
   - Copy variation: `[Angle Name] - [Lead Type] Lead` — e.g., `Quiet Overwhelm - Offer Lead`
   - Visual variation: `[Angle Name] - [Visual Style]` — e.g., `VAM Crew Members - Organic Looking`
   **Keep Angle Names SHORT** (2-4 words, natural casing). The number (#) and CID are auto-generated by the tracker.

6. **Dropdown constraints — use ONLY these values:**
   - Variation Type: **Copy** / **Visual** — ALL creatives in a batch should have the SAME variation type
   - Lead Type: **Offer** / **Promise** / **Problem-Solution** / **Secret** / **Proclamation** / **Story**
   - Awareness Level: **Most Aware** / **Product Aware** / **Solution Aware** / **Problem Aware** / **Unaware**
   - Status: **Ready For Internal**

## Post-Generation Checklist

After generating the batch, verify:
- [ ] **VARIABLE ISOLATION:** All 5 creatives use the EXACT SAME visual style (chosen by AI based on data). Only the tested element (Hook, Body, CTA, etc.) changes across the batch.
- [ ] All 5 creatives have the SAME Variation Type (Copy or Visual)
- [ ] No losing patterns used (check against 04_LOSING_PATTERNS.md)
- [ ] All 5 creatives have different verbal hook types (Copy batches only)
- [ ] Awareness levels are distributed (not all "Most Aware")
- [ ] The AI-chosen visual style is appropriate for the batch focus — ONE style used across all 5
- [ ] Naming follows Creative Tracker formula: Copy var = `[Angle Name] - [Lead Type] Lead`, Visual var = `[Angle Name] - [Visual Style]` (tracker adds # and CID)
- [ ] Copy is specific, not generic — includes numbers, prices, or proof points
- [ ] Each creative has a clear data-driven rationale
- [ ] Variation Type uses ONLY: Copy / Visual
- [ ] Lead Type uses ONLY: Offer / Promise / Problem-Solution / Secret / Proclamation / Story
