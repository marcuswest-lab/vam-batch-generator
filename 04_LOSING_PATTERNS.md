# VAM Confirmed Losing Patterns
*Last updated: March 4, 2026 — Add to this list when ads with $500+ spend show ROAS below 0.5x*

These patterns have been tested with real spend and confirmed as non-performers. **DO NOT recreate ads using these elements.**

---

- **StickyNote (SV29)** — 10 variations, $744 spent, 0 conversions
- **EducationalLayout (SV30)** — 5 variations, $445 spent, 0 conversions
- **HoldingSign (SV19/20/21)** — 6 variations, $266 spent, 0 conversions
- **TruckCalendar (SV22)** — 6 variations, $804 spent, near-zero conversion
- **UsvsThem comparison (SV12)** — 1 variation, $763 spent, 0 sales
- **EmptySpace/Room (SV27)** — 2 variations, $3,269 spent, 0 sales
- **TruckMap (SV28)** — 0 performance
- **Carousel format (SV33 carousel)** — 5 variations, $521 spent, 0 sales
- **Copy-only single image (SV33 single)** — 5 variations, $518 spent, 0 sales
- **ChatGPT conversation (VV2)** — 3 variations, $365 spent, 0 leads
- **Animated truck variations (VV4 white, VV5 Santa, VV6 sledding)** — All losers
- **Single creator IG testimonial (VV13)** — 1 variation, $583 spent, 0 sales

## Losing Geographic Targeting
- **NY-specific copy** — SC47-52 (CheckThisNY, LimitedTime, NYNewPricing, Early26Pricing, NYNewSavings), all losers
- **NY-to-specific-city routes** — SC27-31 (NYtoLA, LAtoAustin, ChicagotoPhoenix, SFtoMiami, SeattletoDallas), zero performance
- **City-specific lifestyle visuals** — SV24_TruckBackNY, SV25_MovingChangesLA, SV26_MovingChangesMia, all losers

## Losing Video Concepts
- **Holiday-themed anything** — Santa hats, sledding, holiday moves (VHK15-18 + VV5/VV6), universally failed
- **Stress-free passive messaging in video** — VHK14_StressFreeMove across all visuals, zero performance
- **"Let us handle" passive messaging** — VHK17/VHK18, zero performance
- **Branded/polished testimonials** — VHK25_Barbara/VV13_IGTestimonial, looks too produced

## Losing Seasonal/Holiday Approaches
- **Holiday V1/V2 CTA variants** — SC21-26 across SV3/SV16/SV17/SV18, 12 variations with ~$2,172 total spend, minimal returns
- **CheckThis Holiday V4/V5** — SC34-35, weak performance despite CheckThis being a winner angle
- **Any creative with "Holiday" in the name** — Pattern: holiday modifiers kill winning angles

## Key Takeaways
1. **Education/information-first visuals don't work** — People don't want to learn about moving, they want to save money
2. **Passive/reassuring messaging fails** — "Relax", "We handle", "Leave it to us" are too soft
3. **Geographic targeting kills performance** — Keep creative nationally focused
4. **Holiday theming kills winning angles** — Even CheckThis (best performer) fails when holiday-ified
5. **Polished/branded production underperforms raw/authentic** — UGC beats studio every time
6. **Carousel format is dead** — Zero performance across all attempts
7. **Comparison/educational layouts generate zero interest** — Moving is emotional, not rational
