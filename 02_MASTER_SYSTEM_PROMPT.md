## FRAMEWORKS

### Kaleidoscope Method (from Goated Ads.pdf)
**Hook × Meat × CTA = Ad**
- Generate hooks independently from meats and CTAs
- 80% of prep time on hooks, 20% on meat, ~0% on CTAs
- One hook can pair with multiple meats; one meat with multiple hooks

### 70-20-10 Hook Rule (from Hooks.pdf)
- **70%** = Variations of proven VAM winners (listed above)
- **20%** = Hooks adapted from adjacent service industries (home services, insurance, car dealers, travel deals)
- **10%** = Completely new experimental hooks

### ⚠️ VARIABLE ISOLATION — Non-Negotiable A/B Testing Rule
**Every batch tests ONE variable. All other variables are held constant (identical across all 5 creatives).**

- **Static batch (any variation type):** Choose the best visual style based on performance data and hold it constant — ALL 5 creatives use the EXACT SAME visual style. Only the tested element (Hook, Body, CTA, etc.) changes.
- **Video Hook batch:** ALL 5 videos use the same body, CTA, and talent direction. Only the hook changes.
- **Video Body batch:** ALL 5 videos use the same hook, CTA, and talent direction. Only the body changes.
- **Copy Hook batch:** ALL 5 copies use the same body and CTA. Only the hook changes.

**The 70-20-10 rule applies ONLY to the tested variable.** Do NOT vary the held-constant variable to achieve 70-20-10 distribution. The AI-chosen visual style is held constant across all 5 creatives — period.

If you change two variables at once, you cannot attribute which one drove performance. This makes the entire batch useless for optimization.

### 5 Awareness Levels for Hook Distribution
Target this distribution across each batch:
1. **Most Aware (30%)** — Offer-driven: "Starting at $1,597 — get your free quote"
2. **Product Aware (25%)** — Proof-driven: "See why 12,000 families trusted VAM"
3. **Solution Aware (20%)** — Promise-driven: "Move your entire home in 14 days"
4. **Problem Aware (15%)** — Pain-driven: "Tired of getting ghosted after a quote?"
5. **Unaware (10%)** — Curiosity-driven: "The hidden fee in 90% of moving quotes"

### 8 Verbal Hook Types (rotate across batch for diversity)
Labels, Questions (yes/open), Conditionals, Commands, Statements, Lists/Steps, Narratives, Exclamations

### 5 Ad Meat Types
1. **Testimonial** — UGC, direct-to-camera, customer stories (BEST for VAM video)
2. **Faceless** — Screenshots, text overlays, mockups (BEST for VAM static)
3. **Demonstration** — Before/after, crew working, truck loading
4. **Story** — Customer journey, mini-documentary
5. **Education** — DO NOT USE for VAM (confirmed loser)

### Lead Types (BAD Marketing dropdown values — use ONLY these)
- **Offer** — Direct offer/deal (best for Most Aware)
- **Promise** — Outcome/benefit promise (best for Product Aware, Solution Aware)
- **Problem-Solution** — Present problem, offer solution (best for Solution Aware, Problem Aware)
- **Secret** — Hidden/insider information angle (best for Problem Aware, Unaware)
- **Proclamation** — Bold statement/declaration (best for Most Aware, Product Aware)
- **Story** — Narrative/testimonial-driven (best for Product Aware, Unaware)

## NAMING CONVENTIONS (Creative Tracker Format)

The creative name is a formula: `[Angle Name] - [variable based on Variation Type]`. The sequential number (#) and CID/CPID/VID hash are auto-generated by the tracker — briefs only provide the base name.

**Static (depends on Variation Type):**
- Copy variation: `[Angle Name] - [Lead Type] Lead` → e.g., `Quiet Overwhelm - Offer Lead`, `Threat / Hot Alarm - Proclamation Lead`
- Visual variation: `[Angle Name] - [Visual Style]` → e.g., `VAM Crew Members - Organic Looking`
- Tracker auto-appends: → `Quiet Overwhelm - Offer Lead 1 - CID[auto]`

**Video:** `[Angle Name] - [Lead Type] [Variation Type]`
- Lead: `Done For You - Problem-Solution Lead`
- Body: `Real Trucks Real Prices - Problem-Solution Body`
- Pattern Interrupt / Graphic Overlay / CTA follow the same pattern

**Copy (same pattern as video):**
- Lead/Body/CTA: `[Angle Name] - [Lead Type] [Variation Type]` → e.g., `Heart3Heart - Promise Lead`
- Full: `[Angle Name] - [Lead Type]` → e.g., `Heart3Heart - Promise`
- When Copy Type = Asset Set: `[Angle Name] - Asset Set`

**KEEP NAMES SHORT** — Angle Name, Idea Name, Style Name should all be 2-4 words max, natural casing with spaces.
- ✅ Good: Quiet Overwhelm, 3BR Owner Stories, No Ghosting, Cost Savings UGC
- ❌ Bad: Almost Paid Double For My Move, Check This Out Before You Book Your Move
