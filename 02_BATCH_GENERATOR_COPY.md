## Instructions

Generate a batch of 5 body copy variations for VAM. Use this when testing new hooks on existing visuals OR new body copy with proven hooks — no new design or video production needed.

## Generation Rules

1. **VARIABLE ISOLATION (critical for proper A/B testing):**
   - **Each batch tests ONE variable. All other variables are held constant.**
   - If **Variation Type = Lead** → ALL 5 copies use the SAME body text and CTA. Only the hook/opening line changes.
   - If **Variation Type = Body** → ALL 5 copies use the SAME hook and CTA. Only the body text changes.
   - If **Variation Type = CTA** → ALL 5 copies use the SAME hook and body. Only the CTA changes.
   - If **Variation Type = Full** → Testing entirely new hook+body+CTA combos (no isolation — use sparingly).
   - The constant elements should be explicitly written out in Copy 1, then clearly referenced as "Same as Copy 1" for subsequent copies. Only the variable being tested gets new content.

2. **Apply 70-20-10 ONLY to the variable being tested (the held-constant variables do NOT change):**
   - Copies 1-3 (70%): Variations of proven CHK/CB combinations
   - Copy 4 (20%): Angle adapted from another industry
   - Copy 5 (10%): Experimental new approach
   - **Important:** 70-20-10 applies to the TESTED variable only. All non-tested components are literally identical across all 5 copies.

3. **Each variation must use a DIFFERENT verbal hook type** (Hook batches) or **different body angle** (Body batches): Labels, Questions, Conditionals, Commands, Statements, Lists/Steps, Narratives, Exclamations

4. **Every variation must be usable with BOTH static and video formats** — write copy that works as image text AND as primary text above an ad

5. **Be specific, not generic** — include numbers, prices, proof points, or specific scenarios. "Save money on your move" is bad. "Save $2,400 on your cross-country move" is good.

6. **Assign proper naming using Creative Tracker formula:** For Lead/Body/CTA: `[Angle Name] - [Lead Type] [Variation Type]` — e.g., `Heart3Heart - Promise Lead`. For Full: `[Angle Name] - [Lead Type]` — e.g., `Heart3Heart - Promise`. When Copy Type = Asset Set: `[Angle Name] - Asset Set`. **Keep Angle Names SHORT** (2-4 words, natural casing). The number (#) and CPID are auto-generated by the tracker.

7. **Dropdown constraints — use ONLY these values:**
   - Variation Type: **Lead** / **Body** / **CTA** / **Full** — ALL creatives in a batch should have the SAME variation type
   - Lead Type: **Offer** / **Promise** / **Problem-Solution** / **Secret** / **Proclamation** / **Story**
   - Awareness Level: **Most Aware** / **Product Aware** / **Solution Aware** / **Problem Aware** / **Unaware**
   - Status: **Ready For Internal**

## Post-Generation Checklist

After generating the batch, verify:
- [ ] **VARIABLE ISOLATION:** Only ONE component changes across all 5 copies. If Hook batch → body and CTA are identical. If Body batch → hook and CTA are identical. If CTA batch → hook and body are identical.
- [ ] All 5 creatives have the SAME Variation Type (all "Lead", all "Body", etc.)
- [ ] No losing copy angles used (no "Licensed & Insured", "Arrived Flawless", etc.)
- [ ] All 5 use different verbal hook types (Hook batches) or different body angles (Body batches)
- [ ] Copy is specific with numbers/prices/proof (not generic)
- [ ] Each variation works for both static and video ad formats
- [ ] Awareness levels are distributed across the batch
- [ ] Headlines are under 40 characters
- [ ] Body copy is 2-4 sentences (not too long)
- [ ] Each has a clear data-driven rationale
- [ ] Variation Type uses ONLY: Lead / Body / CTA / Full
- [ ] Lead Type uses ONLY: Offer / Promise / Problem-Solution / Secret / Proclamation / Story
